**Campus Synergy**

To run this project

git clone https://github.com/Omnaik2021/ByteBusters_CampusSynergy

Install follwong dependencies

npm i express

npm i mongoose

npm i body-parser

npm i path

npm i nodemailer

npm i cookie-parser

npm i hbs

npm i dotenv

To run the project run follwing command in terminal:

npm run dev

Project will be running in port 3000:
http://localhost:3000
